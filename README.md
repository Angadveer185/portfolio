# 📒 Sketchbook Portfolio

A personal portfolio website inspired by the look and feel of a hand-drawn sketchbook.

Instead of a traditional portfolio, this project recreates the experience of flipping through a creative notebook. The interface combines torn paper, doodles, handwritten typography, stickers, and playful animations to showcase my work, projects, and experience.

---

## ✨ Features

* 🎨 Sketchbook-inspired UI
* 📄 Paper-themed navigation and layouts
* ✏️ Hand-drawn doodles and stickers
* 🌊 Smooth scroll-based animations
* 📱 Fully responsive design
* ⚡ Optimized with Next.js
* 🎯 Modular and reusable React components
* 🌙 Modern TypeScript codebase

---

## 🛠️ Tech Stack

* **Next.js**
* **React**
* **TypeScript**
* **Tailwind CSS**

Additional technologies and libraries may be introduced as the project evolves.
